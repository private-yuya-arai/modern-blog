## この知識はいつ使うの？

*   **市場調査**: 全国民にアンケートするのは無理だが、できるだけ少ない人数で正確な「支持率」や「認知度」を知りたいとき。
*   **監査**: 膨大な伝票の中から不正を見つけるために、効率的にチェック対象を選び出したいとき。
*   **実験**: 被験者を集める際に、性別や年代の偏りをなくしたいとき。

## サンプリング手法の選び方チャート

## 各手法の比較

| 手法名 | やり方 | メリット | デメリット | 具体例 |
| :--- | :--- | :--- | :--- | :--- |
| **単純無作為抽出** | 全員からクジ引き | 統計的に一番シンプルで計算が楽。 | 実施コストが高い（全国に散らばる）。 | 全住民からランダムに1000人。 |
| **層化抽出法** | 年代・性別などで層に分け、各層から人数比に合わせて抽出。 | **精度が高い**。重要な層を確実に含められる。 | 層ごとの情報が事前に必要。 | 年代構成比に合わせて、20代〜60代から各人数を選ぶ。 |
| **クラスター抽出法** | 学校や地域（クラスター）を選び、その**全員**を調査。 | **調査が楽**（1箇所に行けば済む）。 | **精度が低い**（同じ場所の人は似がち）。 | ランダムに選んだ「〇〇小学校」の生徒全員を調査。 |
| **多段抽出法** | クラスター抽出して、さらにその中から抽出。 | 現実的な大規模調査のデファクト。 | 計算が複雑になる。 | 市町村を選ぶ $\rightarrow$ 町丁目を選ぶ $\rightarrow$ 世帯を選ぶ。 |

## Pythonでの実装：層化抽出のシミュレーション

ある会社（社員1000人）の意識調査をします。
*   部署A（多数派）：800人、満足度ばらつき大
*   部署B（少数派）：200人、満足度ばらつき小

単純ランダムだと部署Bの人がほとんど選ばれない可能性があります。層化抽出と比較してみましょう。

```python
import numpy as np
import pandas as pd

# 母集団作成
# 部書A: 800人, 平均50, 標準偏差20
dept_A = np.random.normal(50, 20, 800)
# 部書B: 200人, 平均80, 標準偏差5
dept_B = np.random.normal(80, 5, 200)

population = pd.DataFrame({
    'score': np.concatenate([dept_A, dept_B]),
    'dept': ['A']*800 + ['B']*200
})

true_mean = population['score'].mean()
print(f"真の平均満足度: {true_mean:.2f}")

# 1. 単純無作為抽出 (n=100)
sample_simple = population.sample(100)
mean_simple = sample_simple['score'].mean()

# 2. 層化抽出 (各層の比率 8:2 に合わせて抽出)
sample_A = population[population['dept']=='A'].sample(80)
sample_B = population[population['dept']=='B'].sample(20)
sample_strat = pd.concat([sample_A, sample_B])
mean_strat = sample_strat['score'].mean()

print(f"単純抽出の平均: {mean_simple:.2f} (誤差: {mean_simple - true_mean:.2f})")
print(f"層化抽出の平均: {mean_strat:.2f} (誤差: {mean_strat - true_mean:.2f})")
```

何度か実行すると、層化抽出の方が誤差が安定して小さい（分散が小さい）ことがわかります。

## Rでの実装：surveyパッケージ

複雑なサンプリングデータの集計には `survey` パッケージが必須です。重み付け（ウェイト）を考慮して正しい平均や分散を出してくれます。

```r
library(survey)

# 架空の調査データ
# id:人, st:層(strata), wt:ウェイト(逆確率)
data <- data.frame(
  id = 1:5,
  st = c(1, 1, 2, 2, 2),
  val = c(10, 11, 20, 22, 21),
  wt = c(100, 100, 50, 50, 50) # 層1は100人に1人、層2は50人に1人選ばれた
)

# 調査デザインの定義
design <- svydesign(id = ~1, strata = ~st, weights = ~wt, data = data)

# 推定（ウェイトを考慮した平均）
svymean(~val, design)
```

## まとめ

*   全数調査ができないときは**標本調査**。
*   精度を上げたければ、似たもの同士をグループ化して各グループから必ず選ぶ**層化抽出**。
*   コストを下げたければ、塊ごと選ぶ**クラスター抽出**（ただし精度は落ちる）。
*   大規模調査（国勢調査など）では、これらを組み合わせた**多段抽出**が使われる。
